# Selection Statement

Selection statement are used to execute a statement based on a condition. If the given condition is true, then only the statement inside it is executed otherwise the else section is executed.

## if

Syntax:
a. if(condition){
statements(s);
statements(s);
}

    b. if(condition){
        statements(s);
        statements(s);
    } else{
        statements(s);
        statements(s);
    }

    c. if(condition){
        statements(s);
        statements(s);
    } else if(condition){
        statements(s);
        statements(s);
    } else{
        statements(s);
        statements(s);
    }

## switch

Switch statement is used to perform some task according to the choice of the user. The functionality is very similar to if statement.

Switch statement consist of three keywords:

    a. case - To check for the value which has been passed.

    b. break - to break the case after execution of particular choice.
    
    c. default - Statements which will be executed when no choice matches i.e. in case of invalid choice.
