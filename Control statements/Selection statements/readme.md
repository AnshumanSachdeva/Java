# Selection Statement
Selection statement are used to execute a statement based on a condition. If the given condition is true, then only the statement inside it is executed otherwise the else section is executed.
Syntax:
    a. if(condition){
        statements(s);
        statements(s);
    }

    b. if(condition){
        statements(s);
        statements(s);
    } else{
        statements(s);
        statements(s);
    }

    c. if(condition){
        statements(s);
        statements(s);
    } else if(condition){
        statements(s);
        statements(s);
    } else{
        statements(s);
        statements(s);
    }
