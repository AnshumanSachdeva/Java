# Control Statement in Java

Control statements are used to control the flow of execution of statement in a programming language. Usually, all the statements are executed sequentially but if we want to control the flow of execution then we use Control Statement.

## Control Statements are divided into following two categories:

### 1. Selection Statement

a. if
b. switch...case

### 2. Looping Statement

a. while
b. do-while
c. for
